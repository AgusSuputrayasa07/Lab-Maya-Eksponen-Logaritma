Fungsi Eksponensial

Fungsi eksponensial didefinisikan sebagai f(x) = bˣ, di mana b adalah basis (b > 0, b ≠ 1).

f(x) = bˣ  |  b > 0, b ≠ 1

Sifat Utama

Domain: semua bilangan real (-∞, +∞). Range: bilangan positif (0, +∞). Grafik selalu memotong sumbu-Y di titik (0, 1) karena b⁰ = 1.

Asimtot

Grafik mendekati garis y = 0 (sumbu-X) namun tidak pernah menyentuhnya. Garis y = 0 adalah asimtot horizontal.

Pengaruh Nilai b

Jika b > 1: grafik naik (pertumbuhan eksponensial). Jika 0 < b < 1: grafik turun (peluruhan eksponensial). Semakin besar b, semakin curam kurva.

📗
Fungsi Logaritma

Fungsi logaritma adalah invers dari fungsi eksponen. Jika bˣ = y, maka logb(y) = x.

g(x) = logb(x)  |  x > 0

Sifat Utama

Domain: bilangan positif (0, +∞). Range: semua bilangan real (-∞, +∞). Grafik selalu memotong sumbu-X di titik (1, 0) karena logb(1) = 0.

Asimtot

Grafik mendekati garis x = 0 (sumbu-Y) namun tidak pernah menyentuhnya. Garis x = 0 adalah asimtot vertikal.

Logaritma Alami

Basis spesial: bilangan Euler e ≈ 2.718. Disebut logaritma alami dan ditulis ln(x). Sangat berguna dalam pemodelan fenomena alam.

🔗
Hubungan Invers: Cermin Ajaib

Sifat Invers

Fungsi eksponen dan logaritma merupakan fungsi invers satu sama lain. Jika kamu mencerminkan grafik f(x) = bˣ terhadap garis y = x, kamu akan mendapatkan grafik g(x) = logb(x).

b^(log_b(x)) = x
log_b(b^x) = x

Pertukaran Koordinat

Jika titik (a, b) ada di grafik eksponen, maka titik (b, a) ada di grafik logaritma. Contoh: titik (3, 8) ada di grafik f(x) = 2ˣ, maka titik (8, 3) ada di grafik g(x) = log₂(x).

Ini terjadi karena 2³ = 8 dan log₂(8) = 3.
